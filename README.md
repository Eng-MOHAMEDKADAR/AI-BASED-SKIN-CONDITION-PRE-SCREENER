# AI-Based Skin Condition Pre-Screener

A Flutter-based AI application for pre-screening skin conditions using image analysis and machine learning.

## Features
- User authentication
- Image upload for skin analysis
- AI-powered condition detection
- Scan history and results tracking
- Clean and modern UI

## Tech Stack
- Flutter (Dart)
- Firebase Authentication & Firestore
- AI / ML image analysis API

## Getting Started
1. Clone the repository
2. Run `flutter pub get`
3. Configure Firebase
4. Run the app using `flutter run`

## Disclaimer
This application is for educational and pre-screening purposes only and does not replace professional medical diagnosis.
